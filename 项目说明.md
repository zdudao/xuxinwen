# 老许聊实体 - 项目说明

## 这个项目解决什么问题？

作为一个关注实体商业的人，你需要每天花大量时间浏览各种平台的热搜、新闻、行业资讯。

**这个项目帮你：**
- ✅ 每天自动抓取全网热点（不用手动刷11个平台）
- ✅ 筛选出与你相关的商业资讯（招商加盟、店铺动态、消费趋势等）
- ✅ 生成适合朋友圈分享的文案（5分钟完成）
- ✅ 发布到网站供读者查看

### 详细步骤

#### Step 1：启动 RSSHub
```
cd g:\dingyue\rsshub-local
pnpm run dev
```
- 作用：把本地RSS订阅源转换为可访问的格式
- 端口：1200
- 需要运行在后台

#### Step 2：运行 TrendRadar
```
cd g:\dingyue\TrendRadar
python -m trendradar
```
- 自动抓取11个热榜平台
- 自动抓取14个RSS订阅源
- AI自动分类和筛选
- 生成HTML日报

#### Step 3：复制文件
将生成的日报复制到 xuxinwen-static 目录

#### Step 4：发布
推送到 GitHub，自动发布到网站

## 关键配置说明

### AI筛选阈值

在 `config/config.yaml` 中设置 `ai_filter.min_score`：

| 值 | 效果 |
|----|------|
| 0 | 显示所有相关（推荐） |
| 0.3 | 少量过滤 |
| 0.5 | 适中 |
| 0.7 | 严格 |

### 分类标签

系统会自动将新闻分为：
- 招商加盟
- 实体创业
- 社区商业
- 餐饮行业
- 消费趋势
- 供应链
- 门店人力
- 运营创新
